Git101 Ödevi

Localde yaptığımız bir projeyi Git'e yüklemek için aşağıdaki adımları takip edebiliriz;

  1. Terminalde projenin olduğu dizine gidiyoruz,
  2. git clone proje_linki: GitHub'daki projemizi lokale alıyoruz,
  3. git add dosya_adi: Göndereceğimiz dosyayı commit etmeden önce hazırlıyoruz,
  4. git commit -m 'aciklama_metni': add komutu ile hazırladığımız dosyaları bir açıklma metni ile commit ediyoruz,
  5. git push: commit ettiğimiz tüm dosyaları GitHub sunucularına gönderiyoruz.

GitHub'a daha önceden yüklenmiş projeye branch oluşturmak ve kodları yüklemek için aşağıdaki adımları takip edebiliriz;

  1. Terminalde projenin olduğu dizine gidiyoruz,
  2. git clone proje_linki: GitHub'daki projemizi lokale alıyoruz,
  3. git pull: GitHub'daki dosyaları lokale alıyoruz,
  4. git branch branch_adi: Bir isim vererek yeni bir branch oluşturuyoruz,
  5. git checkout branch_adi: Master dalından yeni oluşturduğumuz branch'e geçiyoruz,
  6. git add dosya_adi: Göndereceğimiz dosyayı commit etmeden önce hazırlıyoruz,
  7. git commit -m 'aciklama_metni': add komutu ile hazırladığımız dosyaları bir açıklma metni ile commit ediyoruz,
  8. git push origin branch_adi: Master dalında çalışmadığımız için 'git push' komutu yeterli olmayacaktır. Bu komut ile dosyalarımızı sunucuya gönderiyoruz.
